# csv2json

A command-line utility that quickly and easily converts CSVs to JSON.

## Usage

You can install csv2json using PyPi and use it as follows:

```bash
$ pip install csv2json.py
$ cat <mycsv> | csv2json -p

# See --help for more information.
```
